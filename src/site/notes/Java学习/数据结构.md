## 数组
>数组是一种查询快增删慢的模型
- 查询速度快 通过地址值和索引定位，查询任意数据耗时相同
- 元素在内存中的储存是连续的
- 删除添加效率低 原始数据要前移或后移

## 链表
>链表的每一个元素称为一个结点
>链表中的结点是独立的对象，在内存中中是不连续的，每个节点包含==数据值==和下一个结点的==地址==
- 链表查询慢 无论查哪个数据都要从头开始查找
- 链表的增删比较快 相对于数组
